About DeskFlow

DeskFlow is a help desk ticketing system built as the final project for General Assembly's Software Engineering Immersive bootcamp.

Users can submit support tickets, organize them by priority and status, group them with tags, and discuss them in threaded comments. Each user only sees and manages their own tickets.

Built with

  • Python and Django
  • PostgreSQL
  • HTML, CSS (Flexbox and Grid), and Django templates
  • Deployed on Heroku
